# HLFramework 一键安装文件

#### 介绍
HLFramework是哈林科技自主研发的一套微服务集群式框架体系.install.php提供一键安装哈林框架管理系统的快捷、简易、智能化的安装文件。

#### 软件架构
php + vue 


#### 安装说明

1.  服务器环境 ：linux 服务器 ，最好安装宝塔面板
2.  运行环境：PHP 7.4  ，Nginx 1.18.0  ,mysql 5.6 以上
3.  安装且启用PHP 扩展 SG11，fileinfo
